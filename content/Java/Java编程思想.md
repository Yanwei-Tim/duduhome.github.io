# 一切都是对象 #
##存储在什么地方##

* 寄存器

	最快的存储区，不能控制

* 堆栈

	速度仅次于寄存器，通过堆栈指针获得支持，向下分配内存，向上释放内存。需要知道确切的生命周期，因此灵活性差。

* 堆

	存放所有的Java对象。不需要知道数据的存活时间，因此较为灵活，但是分配和清理内容耗时多。

* 常量存储

	存放在代码内部。

* 非RAM存储

	后续再了解~ 目前没看懂。
